# Finanças para Todos: Guia de Estudo com NotebookLM 💰

## 📋 Sobre o Projeto
Este repositório contém a documentação e os artefatos do desafio de projeto da DIO sobre IA Generativa. O objetivo é criar um ecossistema de aprendizagem ativa no **NotebookLM**, utilizando fontes confiáveis sobre finanças e a metodologia de simplificação financeira difundida por Eduardo Feldberg (Primo Pobre).

## 🧭 Metodologia Guia
A base conceitual deste projeto fundamenta-se nos pilares de:
1. **Pé no chão:** Viver abaixo do que se ganha.
2. **Clareza:** Entender que juros compostos trabalham para você ou contra você.
3. **Acessibilidade:** Traduzir o "economês" para a linguagem do dia a dia.

---

## 📚 Fontes Curadas
Para este estudo, foram selecionadas as seguintes fontes (alinhadas ao pensamento de Feldberg):

1. **Guia de Educação Financeira (BCB):** Material oficial do Banco Central do Brasil sobre gestão de riscos e economia pessoal.
2. **Princípios de Economia Popular:** Artigos sobre como economizar no cotidiano e evitar armadilhas de crédito (Ex: Serasa Ensina).
3. **E-book: O Guia do Poupador Iniciante:** (Material adaptado com base nos vídeos do canal Primo Pobre).
4. **Educação Financeira - ENEF:** Estratégia Nacional de Educação Financeira para brasileiros.

---

## 🛠️ Estrutura do Caderno no NotebookLM
O caderno foi organizado para responder às seguintes perguntas estratégicas:
* *Como montar uma reserva de emergência ganhando um salário mínimo?*
* *Quais são as armadilhas comuns no uso do cartão de crédito e cheque especial?*
* *Como diferenciar ativos de passivos sob a ótica da vida real?*

---

## 🤖 Prompts Utilizados
Nesta seção, registramos os prompts testados para extrair o melhor da IA:

- **Prompt de Resumo:** "Com base nas fontes, explique o conceito de reserva de emergência usando uma analogia simples que qualquer pessoa entenderia, evitando termos técnicos complexos."
- **Prompt de Glossário:** "Identifique os 10 termos mais citados nas fontes e crie um glossário 'traduzido' para a linguagem popular."
- **Prompt de Simulação:** "Crie um passo a passo para sair das dívidas priorizando os juros mais altos, conforme a lógica apresentada nos documentos."

---

## 🚀 Como Reutilizar este Guia
1. Baixe os PDFs listados na pasta `/fontes`.
2. Faça o upload no [NotebookLM](https://notebooklm.google.com/).
3. Utilize os prompts sugeridos no arquivo `prompts.md`.
